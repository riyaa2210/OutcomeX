# Automated-Meeting-Outcome-Tracker
Automated Meeting Outcome Tracker is a backend application that processes meeting audio, transcribes it using AWS Transcribe, and extracts useful outcomes such as summaries, transcripts, and action items.
The project is built using FastAPI and designed to automate post-meeting analysis efficiently.

Tech Stack

Backend: Python, FastAPI

Cloud Services: AWS Transcribe

Note: Local Whisper transcription also requires ffmpeg/ffprobe installed and available on your system PATH when using audio uploads.

Database: SQLite (can be extended to PostgreSQL/MySQL)

API Docs: Swagger (FastAPI built-in)

Version Control: Git & GitHub
#Features :
🎙 Upload meeting audio files

📝 Automatic speech-to-text transcription (AWS Transcribe)

📄 Store meeting transcripts

✅ Extract actionable insights from meetings

🔗 RESTful APIs with FastAPI

📊 Scalable backend architecture
